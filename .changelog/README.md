# Changelog (Legacy)

This directory previously stored pending changelog entries for each release.

The project now uses [release-please](https://github.com/googleapis/release-please) for automated release management. Changelog entries are generated automatically from conventional commit messages on the `main` branch.

See [CHANGELOG.md](../CHANGELOG.md) for the generated changelog and [the release process docs](../docs-site/docs/development/manual-release-process.md) for details.
